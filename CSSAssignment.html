Question 5: What do you understand by CSS opacity?

Answer:

The opacity CSS property controls an element's opacity. The inverse of transparency, opacity, relates to how much of the content behind an element is obscured. Opacity values are always floating numbers, with a minimum of 0 and a maximum of 1. When you use the opacity property to give transparency to an element's background, all of its child elements become translucent as well.

Question 10: Why should background and color be used as separate properties?

Answer

There are two variables at work here: It improves the readability of style sheets. The background property in CSS is a tricky attribute that becomes even more problematic when combined with colour.

The primary distinction between the CSS background and background-color properties is that the background property is an abbreviation for all background attributes. The background-color property, on the other hand, is a subset of the background property, which is used to set the background colour.

Question 13: What are the ways to integrate CSS as a web page?

Answer

There are three ways to include CSS into web sites.

1. The inline approach is used to insert style sheets into an HTML document.

2. Embedded/internal technique - This method is used to apply a distinct style to a single page.

3. Linked/Imported/External approach - This method is utilised when you want to make changes on numerous pages at the same time.

Question 14: What is embedded style sheets?

Answer

CSS that is embedded (internal) is used to apply CSS to a particular document or page. It has the potential to alter all of the page's elements. It is written within the style tag of the HTML head section.

Example:

<style>
p{color:green}
</style>

Question 15: What are the external style sheets?

Answer

An external style sheet is a distinct CSS file that may be accessed by creating a link in the webpage's head section.

CSS is applied to multiple pages or all pages using external CSS. In this section, we write all of the CSS code in a css file. Its extension must be.css, such as style.css.

A CSS selector is the initial component of a CSS Rule. It is a collection of words and patterns that tell the browser which HTML elements to select in order to apply the CSS property values in the rule to them.

Question 18: What are the media types allowed by CSS?

Answer

In media queries, the @media rule is used to apply various styles to different media types/devices. Many things can be checked with media queries, including the width and height of the viewport, as well as the width and height of the device..

Example:

<style tyle = "text/css">
@media print {
body { font-size: 10pt }
}
@media screen {
body { font-size: 12pt }
}
@media screen, print {
body { line-height: 1.2 }
}
</style>
